# ☄️ Halley VBC

Site oficial do **Halley VBC**, um time de vôlei criado por Anthony S. Silva. O projeto tem como objetivo apresentar o time, seus membros, horários, informações e formas de contato.

## 📖 Sobre

O Halley VBC é um time voltado para a amizade, respeito e espírito esportivo. Este site foi desenvolvido para servir como página oficial do grupo, reunindo informações importantes em um só lugar.

## ✨ Funcionalidades

- 🏠 Página Inicial
- ℹ️ Seção Sobre
- 🏐 Informações sobre os treinos
- 📞 Página de Contato
- 📱 Layout responsivo
